# 5.JSP

##  5.1 什么是JSP

  Java  Server  Pages:  Java服务端页面，也和Servlet一样，用动态Web技术！

  最大的特点：

   * 写JSP就像在写HTML
   * 区别：
     * HTML 只给用户提供静态的数据
     * JSP页面中可以嵌入JAVA代码，为用户提供动态数据



## 5.2  JSP原理

   * 服务器内部工作
