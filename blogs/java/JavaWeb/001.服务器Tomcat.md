## 1.3 发布一个web网站

​     **1.将自己写的网站的原码文件，放到服务器(Tomcat) 指定的 web应用的文件夹下，就可以访问了**

​    **网站应该有的结构**

```java
java
 --wepapps :Tomcat服务器的web目录
     -ROOT
     -项目 ：网站的目录名
       -WEB-INF
         -classes : java文件
         -lib : web程序所依赖的Jar包
         -web.xml:网站的配置文件
       -index.html 默认的首页
       -static 静态web文件
             -css
             -js
             -img
       -.........      
```
